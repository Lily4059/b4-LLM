根据 docs/agent_intro.txt 中的信息，Memory 在 Agent 系统中扮演关键角色：

1. **核心定义**：Memory 为 Agent 提供全局知识和历史对话上下文。
2. **系统组成**：Agent 系统由模型、工具、记忆和执行循环组成。
3. **功能作用**：
   - 工具调用让模型能够读取本地文件、执行计算，并把结果用于后续回答。
   - Memory 确保 Agent 能够持续利用历史信息和全局知识，支持连贯的对话和任务执行。

总结：Memory 是 Agent 系统的核心组件之一，负责维护全局知识和历史对话上下文，使 Agent 能够基于过往信息进行更智能、连贯的决策和响应。

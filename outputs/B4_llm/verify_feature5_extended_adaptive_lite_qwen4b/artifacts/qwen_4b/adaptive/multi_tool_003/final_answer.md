这两份文档主要介绍了 Agent 系统的核心概念：

1. **agent_intro.txt**：说明了 Agent 系统由模型、工具、记忆和执行循环组成。工具调用使模型能够读取本地文件、执行计算，并将结果用于后续回答；Memory 为 Agent 提供全局知识和历史对话上下文。

2. **tool_calling.md**：阐述了 Tool Calling 的工作机制——模型负责决定是否调用工具，工具层负责校验和执行，运行时负责维护消息闭环。

**总体主题**：这两份文档共同介绍了 Agent 系统的基本架构和工具调用机制，解释了模型如何通过工具扩展能力，以及工具调用在 Agent 系统中的角色和流程。
